## How to Contribute

1. Fork the repository.
2. Create a new branch: `git checkout -b fix/your-description`
3. Make your changes and commit: `git commit -m 'fix: describe your change'`
4. Push to your fork and open a Pull Request.

## Code Style

- **Java:** Follow standard Java naming conventions (CamelCase for classes).
- **C++:** Use consistent indentation (4 spaces).
- All algorithm implementations must include a comment block explaining
  the time and space complexity.
